# Nono Robot

This repository contains the Arduino code for the Nono robot, a small autonomous robot built on an Arduino Mega.

## Features

*   Differential steering
*   Obstacle avoidance with ultrasonic and laser sensors
*   Compass-based navigation
*   State machine for controlling behavior
*   Serial command interface for control and telemetry
