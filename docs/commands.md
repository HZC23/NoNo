# Commandes Série (Obsolète)

**ATTENTION : L'interface de commande série décrite précédemment (`CMD:<ACTION>:<VALEUR>`) a été dépréciée et n'est plus active dans le firmware actuel.**

Le contrôle du robot s'effectue désormais principalement de deux manières :

1.  **Manette de jeu Xbox (via Bluepad32) :** Pour le contrôle manuel direct.
2.  **Application externe (via protocole JSON) :** Pour le contrôle, la configuration et la télémétrie.

Veuillez vous référer au document [**`communication.md`**](./communication.md) pour une description détaillée des méthodes de contrôle actuelles.
